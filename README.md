# Kusama Yayoi Website

쿠사마 야요이 작가의 작품 세계를 웹으로 표현한 프로젝트입니다.

[Live Demo](https://ji579.github.io/KusamaYayoi/) | [Figma](https://bit.ly/YayoiKusamaWebsiteFigma)

---

## 프로젝트 배경

기존에 분리되어 있던 작가 페이지와 미술관 전시 페이지를 하나의 플랫폼으로 통합했습니다.  
쿠사마 야요이의 대표적인 시각 요소인 반복 패턴과 강렬한 색감을 웹에 적용하면서,  
정보 구조를 정리하고 사용자 동선을 개선하는 데 집중했습니다.

---

## 주요 기능

**페이지 구성**
- 메인 / 작가 소개 / 작품 갤러리 / 전시 정보 페이지
- 각 페이지마다 목적에 맞는 레이아웃 구성

**시각적 표현**
- 작가의 작품에서 추출한 패턴과 색상 활용
- 이미지 중심 레이아웃으로 작품 감상에 집중
- 반복 요소를 활용한 리듬감 있는 화면 구성

**반응형 디자인**
- PC, 태블릿, 모바일 각각의 화면에 최적화
- 이미지 비율과 텍스트 배치를 디바이스별로 조정

**인터랙션**
- 스크롤 기반 페이드인 효과
- 호버 시 이미지 확대 및 정보 표시
- 페이지 전환 시 부드러운 트랜지션

---

## 기술 스택

**HTML5**  
시맨틱 태그로 페이지 구조를 명확하게 구분했습니다.  
작가 소개, 작품 목록, 전시 정보 등 각 섹션의 역할이 마크업에 드러나도록 작성했습니다.

**CSS3**  
Flexbox와 Grid로 레이아웃을 구성했습니다.  
작품 이미지 배치, 여백 조절, 색상 조합에 신경 썼고,  
쿠사마 야요이 작품의 특징인 반복 패턴을 CSS로 구현했습니다.

**JavaScript (ES6)**  
이미지 슬라이더, 스크롤 애니메이션, 모달 팝업 등을 구현했습니다.  
jQuery 없이 바닐라 자바스크립트로 작성했고, 필요한 부분만 DOM을 조작하도록 최적화했습니다.

## 구현 세부사항

### 1. 통합 네비게이션
기존에는 작가 페이지와 전시 페이지가 별도로 운영되어 정보를 찾기 어려웠습니다.  
전체 콘텐츠를 하나의 사이트 구조로 정리하고, 일관된 네비게이션을 적용했습니다.

### 2. 작품 갤러리
작품 이미지를 그리드로 배치하고, 클릭하면 확대해서 볼 수 있도록 했습니다.  
이미지 로딩 속도를 고려해 lazy loading을 적용했습니다.

### 3. 색상 시스템
쿠사마 야요이 작품의 색상 팔레트를 추출해서 CSS 변수로 정의했습니다.  
전체 사이트에서 일관된 색감을 유지하면서도 페이지마다 강조 색상을 다르게 사용했습니다.

```css
:root {
  --primary: #FF0000;
  --secondary: #FFD700;
  --accent: #000000;
  --bg: #FFFFFF;
}
```

### 4. 반응형 이미지 처리
다양한 화면 크기에 대응하기 위해 `srcset`과 `picture` 태그를 사용했습니다.  
모바일에서는 가벼운 이미지를, 데스크톱에서는 고해상도 이미지를 로드합니다.

### 5. 접근성
스크린 리더 사용자를 위한 대체 텍스트를 모든 이미지에 추가했습니다.  
키보드만으로도 모든 콘텐츠에 접근할 수 있도록 포커스 스타일을 적용했습니다.

## 개선 과정

**초기 문제**
- 정보가 여러 곳에 흩어져 있어 찾기 어려움
- 일관성 없는 디자인
- 모바일 최적화 부족

**해결 방법**
- 콘텐츠를 논리적으로 재구성
- 작가의 시각적 스타일을 기반으로 디자인 시스템 구축
- 모바일 우선 접근 방식으로 레이아웃 설계

## 프로젝트 구조

```
KusamaYayoi/
├── index.html              # 메인 페이지
├── about.html              # 작가 소개
├── works.html              # 작품 갤러리
├── exhibitions.html        # 전시 정보
├── css/
│   ├── reset.css          # 브라우저 기본 스타일 초기화
│   ├── common.css         # 공통 스타일
│   └── pages.css          # 페이지별 스타일
├── js/
│   ├── main.js            # 메인 스크립트
│   └── gallery.js         # 갤러리 기능
└── images/                # 이미지 파일
```

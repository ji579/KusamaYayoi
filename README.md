# KusamaYayoi
프로젝트 개요

Kusama Yayoi 공식 웹사이트 또는 전시 콘텐츠를 참고해

작가의 작품 세계관과 시각적 특징을 이해하고,

브랜드 감성과 예술적 요소를 퍼블리싱 요소로 재해석해 구현한 웹 퍼블리싱 프로젝트입니다.


공식 자료는 쿠사마 야요이의 설치 작품, 전시 정보, 작가 소개 등으로 구성되어 있으며,

이를 참고해 사이트의 전체 페이지 구조와 콘텐츠 흐름을 분석했습니다.

본 프로젝트에서는 작품이 가진 반복적 패턴과 몰입형 비주얼 감각을 웹 화면에서 반영하며,

단순한 복제가 아닌 레이아웃 구성, 비주얼 요소 표현, 인터랙션 적용을 통해

작가 고유의 예술적 특성을 웹 퍼블리싱으로 풀어낸 것이 특징입니다.


실현 기능

다중 페이지 구조 구현
(메인 페이지 + 작가 소개 / 작품 소개 / 전시 정보 등 페이지 별 레이아웃 구성)
브랜드·작가 감성 반영
쿠사마 야요이 작품의 반복적 패턴과 몰입형 요소를 비주얼로 표현
이미지 중심 레이아웃 구현
작품·전시 이미지 혼합 구성 및 시각적 흐름 최적화
시맨틱 마크업 기반 구조 설계
의미 기반의 HTML 태그로 콘텐츠 구조 분리 및 가독성 향상
CSS 레이아웃 활용
Flexbox, Grid 등을 통한 유기적 구성과 여백/정렬 제어
기본 인터랙션 적용
hover 또는 스크롤 기반 효과로 시각적 반응 구현

사용 기술

HTML5
시맨틱 구조로 페이지별 역할을 명확히 구분하고,
작가/전시/정보 페이지를 의미 기반으로 설계
CSS3
Flexbox/그리드 레이아웃으로 구성하며, 쿠사마 작품의 감성과 조화를 이루도록
이미지 박스, 텍스트 비율, 색상·여백 등을 세밀하게 조절
JavaScript (ES6)
페이지 내 이미지/인터랙션 제어, 모션 요소, 동적 스타일 적용 등
사용자 체험 요소 구현
